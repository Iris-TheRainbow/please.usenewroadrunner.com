PLEASE use Roadrunner 1.0

Road Runner 0.5 has now been replaced by Roadrunner 1.0 for over a year. Roadrunner 1.0 has some major new features like a new commandbase like Actions system, improved tuning, improved path following, and is generally just.. better.

If you have used the guide at learnroadrunner.com, you are using Road Runner 0.5. The offical documentation for Road Runner 1.0 can be found rr.brott.dev


The major differences include:

1. A HUGE overhaul to the tuning process, making it much faster overall and much less buggy

2. Speaking of bugs, theres far fewer. No longer does Road Runner just.. not work sometimes.

3. An entire commandbase-like system to replace markers.


You can find more information about the differences on our migration page
